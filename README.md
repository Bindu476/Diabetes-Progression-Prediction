# 🩺 Diabetes Progression Prediction

## 📌 Overview
This project predicts diabetes disease progression after one year using patient health measurements from the Scikit-Learn Diabetes Dataset. A Linear Regression model is used to analyze and predict disease progression scores.

## 🎯 Features
- Data Loading and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Linear Regression Model
- Model Evaluation (R², MAE, RMSE)
- Feature Importance Analysis
- Actual vs Predicted Visualization

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## 📊 Dataset
- Source: Scikit-Learn Diabetes Dataset
- Samples: 442
- Features: 10
- Target: Diabetes Progression Score

## 🚀 Results
The model predicts diabetes progression and evaluates performance using:
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
